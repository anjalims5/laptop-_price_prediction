# laptop-_price_prediction
# 💻 Laptop Price Prediction using Machine Learning

This project predicts laptop price levels based on hardware specifications using machine learning techniques. A simple and interactive Streamlit UI is included for real-time price predictions.

---

## 📊 Algorithms Used

- Decision Tree Regression  
- Support Vector Regression (SVR)  
- ✅ Random Forest Regression (Best Performing)

---

## 🏆 Best Model

The Random Forest Regressor achieved the highest accuracy and was selected as the final model for price prediction.

---

## 🌐 Streamlit Web App

An interactive UI was created using Streamlit to allow users to input laptop specifications and get an instant price prediction.

---

## ▶ How to Run the App

1. Make sure all required libraries are installed (see below).  
2. Open the project in your code editor (e.g., VS Code).  
3. Run the following command in your terminal:

```bash
